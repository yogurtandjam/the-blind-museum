description: The Blind Museum's core concept, goals, and development direction as of March 2026
type: project
---

Core concept: a web museum where artwork is only visible when the user's eyes are closed (detected via TensorFlow.js face mesh). Currently searches the Met Museum API and displays image results.

**Why:** The goal is maximum immersion — making the user *feel* like they're in a museum, not just browsing images. The eyes-closed mechanic is both the accessibility hook and the core interaction.

**How to apply:** Feature ideas should be evaluated against two criteria: (1) does it deepen immersion? (2) does it preserve the eyes-closed constraint? The user is interested in spatial traversal (walking through Met halls) but open to enhancing the current experience if 3D assets aren't available.
